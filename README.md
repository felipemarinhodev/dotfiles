# Para instalar.
<ol>
	<li>Baixe o repositorio</li>
	<li>Entre na pasta do repositorio pelo terminal.</li>
	<li>Executar o comando: Chmod +x install.sh </li>
	<li>Executar o comando: ./install.sh </li>
</ol>
